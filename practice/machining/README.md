# 机械制造技术基础客观题刷题包（可分享版）

这是可分享版，已移除近期两份不可外传试卷，同时移除了题库中来源于这两份试卷的全部题目与隔离记录。

当前题库共 220 题：选择 111 题、判断 83 题、填空 26 题。

包含来源：

- `18-19.pdf`
- `19-20.pdf`
- `21B.pdf`
- `22A.pdf`
- `23A.pdf`

## 使用方法

1. 双击 `start-practice-server.bat`。
2. 等浏览器自动打开刷题页面。
3. 不要关闭弹出的命令窗口；关闭后本地服务会停止。

如果浏览器没有自动打开，复制命令窗口里显示的 `http://127.0.0.1:端口/site/index.html` 地址到浏览器。

## 文件说明

- `start-practice-server.bat`：双击启动入口。
- `site/index.html`：交互刷题页面。
- `site/interactive_questions.json`：可分享版题库数据。
- `site/quarantined_questions.json`：隔离题库；可分享版当前为空。
- `*.pdf`：可分享原卷，用于页面里的“回原卷”按钮。

## 分享前检查

本版本不包含近期禁传试卷、试卷截图缓存或临时导出文本。
